## Simple and Easy guide to stylize your MyAnimeList anime & manga style the way you want

### On https://myanimelist.net/ownlist/style select both Anime and Manga style to Dark Green
![image](https://user-images.githubusercontent.com/35408292/126910881-7ae5c842-5c4a-485b-853c-c39e40e0d6fb.png)

### Now to edit the css click on the image card, it will redirect you to the editor<br>

### Editor Example <br>
![image](https://user-images.githubusercontent.com/35408292/126910945-41053e8c-c8ea-4365-8a80-87863d9203e3.png)

### Select the radio button and upload an image if you want a background <br>
![image](https://user-images.githubusercontent.com/35408292/126910966-640b801b-8f3c-46c7-88af-79ad7d315312.png)

### Configure some background properties <br>
![image](https://user-images.githubusercontent.com/35408292/126911013-517edb80-cc74-4a57-9d04-173351506a98.png)

### Paste the css content from the file https://github.com/DouglasKosvoski/MAL-custom-style/blob/main/darkgreen.css on the designed area. Or create your own style <br>
![image](https://user-images.githubusercontent.com/35408292/126911024-324a11f6-d4fa-416b-be98-a85e41f57abb.png)

### Finally hit save and its all done. You can now close this tab <br>
![image](https://user-images.githubusercontent.com/35408292/126911063-3b0825f5-a921-4b3b-9e74-429ace8669ed.png)
